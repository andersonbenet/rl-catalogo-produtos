# RL Catálogo de Produtos

Catálogo web da RL Artigos para Pesca.

## Estrutura

```
rl-catalogo-produtos/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos
├── assets/
│   ├── logo/           # Logo da empresa
│   └── produtos/       # Imagens dos produtos (produto-001.png, ...)
└── README.md
```

## Como usar

Abra `index.html` no navegador ou publique em GitHub Pages.
